# Meus Primeiros Passos com Flowgorithm

Olá S2! Eu sou a Livia, Seja bem-vindo ao meu repositório de lógica de programação. Aqui você encontrará uma série de exercícios desenvolvidos utilizando o Flowgorithm, com o objetivo de desenvolver o raciocínio lógico por meio da criação de fluxogramas.

## Atividades Desenvolvidas
A seguir estão cinco atividades de lógica de programação, começando com operações simples e avançando para estruturas condicionais.
